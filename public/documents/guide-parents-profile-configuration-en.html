üìã STEP 1: PARENT PROFILE CREATION

1.1 Initial Registration

  1. Go to https://www.educafric.com
  2. Click "Create Account"
  3. Select "Parent/Guardian"
  4. Fill in basic information:
    • First and last name
    • Personal email (required)
    • Secure password
    • Primary phone number

1.2 Family Information

  1. Relationship: Father, Mother, Legal Guardian, Other
  2. Home Address: Family's main address
  3. Emergency Phone: Secondary contact number
  4. Profession: Optional information
  5. Communication Preferences: SMS, WhatsApp, Email

üë∂ STEP 2: CONNECTING WITH CHILDREN

2.1 Adding Your Children

  1. In your dashboard, click "Add a Child"
  2. Fill in the information:
    • Child's first and last name
    • Date of birth
    • School attended
    • Class/Level
    • Student ID (if available)
  3. Wait for school validation
  4. Repeat for each child

2.2 Validation and Activation

  1. The school will receive a validation request
  2. You will receive a confirmation notification
  3. Once validated, full access to tracking
  4. Configuration of personalized settings
